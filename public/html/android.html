Downloading and Setting Up Crew Link on Android

Step One

Visit the Google Play Store and install BetterCrewlink.
Voice chat will only work if a PC user is hosting the game with 'Mobile Host' checked in BetterCrewlink settings

Step Two

Select option Voice Server and set to Custom server. Replace the default server with

Copy Server URL

Step Four

Set your Ingame name to match your Among Us name exactly, each name must be unique.
Enter the Among Us lobby code and select your mic. The first option should do it.

Step Four

Press Connect then launch and join your Among Us lobby. You may need to switch back to BetterCrewLink to refresh your connection if you have voices issues.

You may need to disconnect if you are kicked out the game. The lobby code MUST be updated any time a new lobby is created.
